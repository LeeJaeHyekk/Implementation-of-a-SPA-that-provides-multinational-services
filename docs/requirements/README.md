# 요구사항 문서

이 폴더에는 프로젝트의 요구사항 관련 문서가 포함되어 있습니다.

## 📄 문서 목록

### 요구사항 체크리스트
- [`checklist.md`](./checklist.md): 프로젝트 요구사항 체크리스트
  - 필수 요구사항
  - 선택 요구사항
  - 구현 상태 확인

## 📝 문서 작성 가이드

새로운 요구사항 문서를 추가할 때는 다음 규칙을 따르시기 바랍니다:

1. 파일명은 `kebab-case.md` 형식을 사용합니다.
2. 요구사항은 명확하고 측정 가능하게 작성합니다.
3. 구현 상태를 체크리스트 형식으로 관리합니다.
