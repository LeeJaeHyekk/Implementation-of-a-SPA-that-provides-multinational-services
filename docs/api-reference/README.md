# API 레퍼런스

이 폴더에는 프로젝트의 소스 코드 관련 문서가 포함되어 있습니다.

## 📄 문서 목록

### 상표 엔티티 라이브러리
- [`trademark-lib.md`](./trademark-lib.md): 상표 데이터 처리 관련 유틸리티 모듈 문서
  - 파싱 및 정규화
  - 필터링
  - 전처리
  - 매칭 로직

### 검색 기능 라이브러리
- [`search-lib.md`](./search-lib.md): 검색어 처리 및 필터링 관련 유틸리티 모듈 문서
  - 검색어 정제
  - 검색어 정규화
  - 검색어 매칭
  - 검색어 검증
  - 필터 결합

### 레거시 코드
- [`pages-legacy.md`](./pages-legacy.md): Pages Router 레거시 코드 안내
  - App Router로 마이그레이션된 레거시 코드 설명
  - 현재 라우팅 구조

## 📝 문서 작성 가이드

새로운 API 문서를 추가할 때는 다음 규칙을 따르시기 바랍니다:

1. 파일명은 `kebab-case.md` 형식을 사용합니다.
2. 문서는 해당 모듈의 구조, 주요 기능, 사용 예시를 포함해야 합니다.
3. 코드 예시는 TypeScript를 사용합니다.
