# 리소스 문서

이 폴더에는 프로젝트의 리소스(이미지, 정적 파일 등) 관련 문서가 포함되어 있습니다.

## 📄 문서 목록

### Assets 이미지
- [`assets-images.md`](./assets-images.md): 소스 코드에서 직접 import해서 사용하는 이미지 파일 안내
  - TypeScript/React에서 import 방법
  - Next.js Image 컴포넌트 사용법
  - 경로 규칙

### 정적 이미지
- [`public-images.md`](./public-images.md): Next.js의 `public/` 디렉토리 내 정적 이미지 파일 안내
  - HTML/JSX에서 사용 방법
  - CSS에서 사용 방법
  - Next.js Image 컴포넌트 사용법

## 📝 문서 작성 가이드

새로운 리소스 문서를 추가할 때는 다음 규칙을 따르시기 바랍니다:

1. 파일명은 `kebab-case.md` 형식을 사용합니다.
2. 문서는 리소스의 위치, 사용 방법, 경로 규칙을 포함해야 합니다.
3. 코드 예시를 포함하여 실제 사용법을 명확히 설명합니다.
