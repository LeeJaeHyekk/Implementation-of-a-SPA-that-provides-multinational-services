# 기능 문서

### 국가별 데이터 비교 기능
- [`comparison-plan.md`](./comparison-plan.md): 국가별 데이터 비교 기능 계획 문서
  - 기능 요구사항

  - 기술적 고려사항

- [`comparison-implementation.md`](./comparison-implementation.md): 국가별 데이터 비교 기능 구현 예시

  - 코드 예시
  - 사용 가이드

## 📝 문서 작성 가이드

새로운 기능 문서를 추가할 때는 다음 규칙을 따르시기 바랍니다:

1. 파일명은 `kebab-case.md` 형식을 사용합니다.
2. 기능 계획 문서는 요구사항, 설계, 구현 계획을 포함해야 합니다.
3. 구현 문서는 실제 코드 예시와 사용 방법을 포함해야 합니다.
