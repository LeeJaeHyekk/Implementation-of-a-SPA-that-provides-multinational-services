## 📝 문서 작성 가이드

새로운 개발 문서를 추가할 때는 다음 규칙을 따르시기 바랍니다:

1. 파일명은 `kebab-case.md` 형식을 사용합니다.
2. 문서는 배경, 목표, 과정, 결과를 포함해야 합니다.
3. 기술적 결정에 대한 근거를 명확히 설명합니다.
